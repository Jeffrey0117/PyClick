功能特色

強大的圖像識別能力，搭配彈性的操作模式，滿足各種自動化需求

🎯

智能圖像識別

採用 OpenCV 模板匹配技術，精準找到螢幕上與模板相似的目標區域，相似度門檻可調整。

🔥

自動偵測模式

持續掃描螢幕，一旦發現目標就自動點擊。支援可調整的掃描間隔（0.3 ~ 2 秒）。

⌨️

熱鍵觸發模式

按下 F6 手動觸發找圖點擊，適合需要人工控制時機的場景。

👆

無焦點點擊

點擊時不會搶走前景視窗的焦點，你可以繼續使用其他應用程式而不受干擾。

⚡

瞬間點擊模式

提供瞬間點擊與穩定點擊兩種模式，瞬間模式無延遲、穩定模式確保點擊被正確識別。

🔄

連續點擊功能

開啟連續點擊後，只要目標存在就會持續點擊，適合需要快速連點的情境。

💾

模板管理系統

儲存多個點擊模板，可設定預設模板，啟動時自動載入，快速切換不同任務。

🔵

藍色區域偵測

內建藍色區域偵測功能，快速標記螢幕上的藍色元素，方便框選目標。

📊

點擊統計功能

即時統計已完成的點擊次數，讓你掌握自動化作業的進度。

使用說明

三個簡單步驟，立即開始自動化點擊

1

截圖並框選目標

點擊「截圖」按鈕擷取當前螢幕，然後在預覽區域中拖曳框選你想要點擊的目標。建議選取範圍適中，太小容易誤判、太大會影響效能。可使用滾輪縮放、Alt+拖曳移動來精確定位。

2

儲存模板

框選完成後，點擊綠色的「儲存選取」按鈕將目標區域存為模板。你也可以使用「測試找圖」功能，確認模板能正確識別目標。

3

選擇模式並啟動

選擇「自動點擊」模式，程式會持續掃描螢幕並自動點擊目標；或選擇「熱鍵模式」，按 F6 手動觸發。啟動自動模式後視窗會自動縮小到系統托盤，避免影響點擊作業。

A

自動模式

持續掃描螢幕，發現目標自動點擊。適合需要長時間執行的自動化任務。

H

熱鍵模式

按下 F6 時觸發一次找圖點擊。適合需要人工控制執行時機的場景。

O

關閉模式

停止所有自動操作。程式仍在背景運行，可隨時切換至其他模式。

下載與安裝

使用 Python 直接執行，或打包成獨立執行檔

從原始碼執行

確保已安裝 Python 3.8 以上版本

# 複製專案
git clone https://github.com/your-repo/pyclick.git
cd pyclick

# 安裝相依套件
pip install -r requirements.txt

# 執行程式
python tray_clicker.py

系統需求

  • Windows 10 / 11
  • Python 3.8+
  • 相依套件：opencv-python, numpy, mss, pyautogui, pystray, keyboard, Pillow
