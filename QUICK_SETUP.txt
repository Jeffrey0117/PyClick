📍 第一步：設定 About 描述
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 前往：https://github.com/Jeffrey0117/PyClick

2. 在頁面右側找到 "About" 區塊（在 README 上方）

3. 點擊 About 右上角的 ⚙️ 齒輪圖示

4. 在彈出的對話框中填寫：

   Description:
   ┌────────────────────────────────────────────────────────┐
   │ 🎯 Windows 智能自動點擊器 - 基於圖像識別的自動化工具    │
   │ 看到圖→點過去→按按鍵 | OpenCV | Script system         │
   └────────────────────────────────────────────────────────┘


📍 第二步：添加 Topics 標籤
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

在同一個對話框中，"Topics" 輸入框中依次添加：

必須添加（7個核心標籤）：
  ✓ automation
  ✓ autoclicker
  ✓ auto-clicker
  ✓ opencv
  ✓ image-recognition
  ✓ python
  ✓ windows

建議添加（10個輔助標籤）：
  • template-matching
  • gui-automation
  • productivity
  • desktop-automation
  • windows-automation
  • opencv-python
  • pyautogui
  • tkinter
  • system-tray
  • testing

輸入方式：輸入關鍵字 → 按 Enter → 重複


📍 第三步：儲存設定
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

點擊對話框底部的 "Save changes" 按鈕

💡 為什麼需要手動設定？

GitHub Repository 的 About 和 Topics 是網頁設定，不是代碼文件，
我無法通過 git commit 直接修改。需要你登入 GitHub 網頁操作。
