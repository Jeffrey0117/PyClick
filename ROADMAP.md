# PyClick 功能路線圖

> 未來可開發的功能整理，依優先級和複雜度分類

---

## 📌 優先開發（實用性高、實作簡單）

### 1. 區域限制搜尋
**複雜度：低 | 價值：高**

只在指定的螢幕區域搜尋模板，而非全螢幕。

- 提升搜尋速度（區域越小越快）
- 減少誤判（避免匹配到其他視窗的相似圖案）
- 實作方式：截圖時裁切指定區域

```
用途範例：
- 只監控遊戲視窗內的按鈕
- 只偵測右下角的通知
```

### 2. 等待模板出現
**複雜度：低 | 價值：高**

暫停執行，直到指定圖片出現才繼續。

- 設定等待逾時（例如最多等 30 秒）
- 可設定掃描間隔
- 逾時後可選擇：跳過 / 停止 / 執行備用動作

```
用途範例：
- 等待載入畫面結束
- 等待彈窗出現再點擊
```

### 3. 等待模板消失
**複雜度：低 | 價值：高**

等待指定圖片從畫面消失後才繼續。

```
用途範例：
- 確認對話框已關閉
- 確認載入動畫結束
```

### 4. 相對座標點擊
**複雜度：低 | 價值：中**

點擊模板位置的偏移處，而非正中央。

- 設定 X/Y 偏移量（像素）
- 支援百分比偏移（模板寬高的 %）

```
用途範例：
- 找到標題，點擊旁邊的關閉按鈕
- 找到輸入框標籤，點擊右側的輸入區
```

## 🔄 流程控制（中等複雜度）

### 5. 腳本串接
**複雜度：中 | 價值：高**

依序執行多個腳本，形成工作流程。

- 腳本 A 完成 → 腳本 B → 腳本 C
- 支援條件跳轉（成功/失敗執行不同腳本）
- 可設定重複次數

```json
{
  "workflow": [
    { "script": "login", "on_fail": "stop" },
    { "script": "collect_reward", "repeat": 5 },
    { "script": "logout" }
  ]
}
```

### 6. 條件分支
**複雜度：中 | 價值：高**

根據畫面狀態執行不同動作。

- IF 圖片 A 存在 → 執行動作 A
- ELSE IF 圖片 B 存在 → 執行動作 B
- ELSE → 執行預設動作

```
用途範例：
- 偵測到「確定」按鈕就點擊，偵測到「取消」就按 ESC
- 根據不同彈窗類型做不同處理
```

### 7. 循環執行
**複雜度：低 | 價值：中**

重複執行當前腳本 N 次或無限循環。

- 設定重複次數（1-999 或無限）
- 設定每輪之間的等待時間
- 可設定停止條件（例如某圖片出現就停）

## 🖱️ 動作擴展

### 8. 右鍵點擊
**複雜度：低 | 價值：中**

支援滑鼠右鍵點擊。

- 動作選項：左鍵 / 右鍵 / 中鍵
- 用於開啟右鍵選單

### 9. 拖曳動作
**複雜度：中 | 價值：中**

從起點拖曳到終點。

- 方式一：從模板 A 拖到模板 B
- 方式二：從模板位置拖曳指定距離
- 設定拖曳速度

```
用途範例：
- 拖曳滑桿
- 拖曳物品到指定位置
```

### 10. 鍵盤輸入文字
**複雜度：低 | 價值：中**

點擊後輸入指定文字。

- 支援一般文字輸入
- 支援特殊鍵組合（Ctrl+V 等）
- 可設定輸入速度（模擬人類打字）

```
用途範例：
- 自動填寫表單
- 輸入搜尋關鍵字
```

### 11. 滾輪捲動
**複雜度：低 | 價值：低**

在指定位置執行滾輪捲動。

- 設定捲動方向（上/下）
- 設定捲動量

## 📊 監控與除錯

### 12. 即時日誌視窗
**複雜度：中 | 價值：高**

在 GUI 內顯示執行日誌。

- 顯示最近 N 條日誌
- 不同等級用不同顏色（INFO/WARN/ERROR）
- 可暫停/清除日誌

### 13. 匹配預覽視窗
**複雜度：中 | 價值：高**

即時顯示在哪裡找到匹配。

- 小視窗顯示當前畫面
- 用紅框標示匹配位置
- 顯示匹配信心度

```
用途範例：
- 除錯為什麼沒匹配到
- 確認匹配位置是否正確
```

### 14. 統計儀表板
**複雜度：中 | 價值：低**

顯示詳細統計資訊。

- 執行次數、成功率
- 平均匹配時間
- 歷史記錄圖表

---

## ⏰ 排程與自動化

### 15. 延遲啟動
**複雜度：低 | 價值：中**

點擊開始後，等待 N 秒才真正開始執行。

- 倒數計時顯示
- 給使用者切換視窗的時間

### 16. 定時執行
**複雜度：中 | 價值：中**

設定在指定時間自動執行。

- 每天固定時間執行
- 每隔 N 小時執行一次
- 支援多個排程

### 17. 間歇執行
**複雜度：低 | 價值：中**

執行一段時間後休息，然後繼續。

- 執行 X 分鐘 → 休息 Y 分鐘 → 重複
- 防止長時間連續操作

## 🔧 進階功能（複雜度高）

### 18. OCR 文字識別
**複雜度：高 | 價值：高**

識別畫面上的文字並觸發動作。

- 使用 Tesseract 或 PaddleOCR
- 匹配指定文字時觸發
- 可提取文字內容供後續使用

```
用途範例：
- 偵測到「錯誤」文字時停止
- 讀取數字並判斷條件
```

### 19. 錄製模式
**複雜度：高 | 價值：中**

錄製使用者操作，自動產生腳本。

- 記錄滑鼠點擊位置和時機
- 記錄鍵盤輸入
- 自動截圖作為模板

### 20. API 遠端控制
**複雜度：高 | 價值：低**

透過 HTTP API 控制 PyClick。

- 啟動/停止/切換腳本
- 取得執行狀態
- 適合整合到其他自動化系統
